# Kshana Landing Website

A modern, responsive landing page for the Kshana messaging app with real-time visitor tracking.

## Features

### 🎨 Design
- Modern gradient UI matching the app theme
- Fully responsive (mobile, tablet, desktop)
- Smooth animations and transitions
- Professional color scheme with primary (indigo) and secondary (pink) colors

### 📊 Visitor Tracking
- Real-time visitor counter in navbar
- Detailed visitor statistics:
  - Total visitors
  - Today's visitors
  - This week's visitors
  - This month's visitors
- Recent visitor timeline
- Local storage persistence
- Export visitor data as JSON

### 📱 Sections
1. **Navigation** - Sticky navbar with visitor count
2. **Hero** - Eye-catching headline with CTA buttons
3. **Features** - 8 key app features with icons
4. **Screenshots** - App UI mockups (placeholder for real screenshots)
5. **Download** - Multi-platform download buttons (iOS, Android, Web)
6. **Stats** - Visitor analytics and timeline
7. **Footer** - Links and social media

   ```javascript
   };
   ```

3. Customize social links in `index.html` footer

## Visitor Tracking

### How It Works
- Each visitor is tracked with:
  - Unique ID
  - Timestamp
  - Date and time
  - User agent (browser info)
  - Referrer source

- Data is stored in browser's localStorage
- Updates in real-time
- Persists across sessions

### Analytics Commands

Open browser console and use:

```javascript
// Get detailed analytics
getAnalytics()

// Export visitor data as JSON
exportVisitorData()

// Get raw stats
tracker.getVisitorStats()
```

# Kshana Website - Quick Setup Guide

## 🚀 Getting Started

### Step 1: Replace Placeholder Images

The website currently uses SVG placeholders. Replace them with real app screenshots:

#### In `index.html`:

**Hero Section (Line ~60):**
```html
<!-- Replace this -->
<img src="data:image/svg+xml,..." alt="App Screenshot">

<!-- With this -->
<img src="images/hero-screenshot.png" alt="Kshana App Screenshot">
```

**Screenshots Section (Lines ~180-210):**
```html
<!-- Replace each screenshot -->
<img src="data:image/svg+xml,..." alt="Chat Screen">

<!-- With real images -->
<img src="images/chat-screen.png" alt="Chat Screen">
<img src="images/stories-screen.png" alt="Stories">
<img src="images/profile-screen.png" alt="Profile">
<img src="images/notifications-screen.png" alt="Notifications">
```

### Step 2: Update Download Links

Edit `script.js` (around line 120):

```javascript
const downloadLinks = {
    ios: 'https://apps.apple.com/app/kshana',      // Your App Store link
    android: 'https://play.google.com/store/apps/details?id=com.kshana',  // Your Play Store link

};
```

### Step 3: Update Social Links

In `index.html` footer (around line 280):

## 🐛 Troubleshooting

### Images not showing
- Check file paths are correct
- Ensure images are in correct folder
- Verify image formats are supported

### Visitor count not updating
- Check browser localStorage is enabled
- Clear cache and reload
- Check browser console for errors

### Download buttons not working
- Verify URLs are correct
- Test links in new tab
- Check for CORS issues

### Styling looks broken
- Clear browser cache
- Hard refresh (Ctrl+Shift+R)
- Check CSS file is loading
- Verify no CSS conflicts
